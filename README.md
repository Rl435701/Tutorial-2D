- CPU: Mínimo 4 núcleos.

- RAM: 8 GB (mínimo), 16 GB (recomendado).

- GPU: Dedicada (NVIDIA o AMD) con al menos 2 GB de VRAM.

- Sistema Operativo: Windows 10/11 de 64 bits.
